# Exercice JS #1 - Manipulation du DOM

## Sujet

Via l'utilisation de la manipulation du DOM, réaliser une application permettant à l'utilisateur d'ajouter, à un listing de Todo, des tâches à acomplir. Les tâches comporteront un titre, une date d'échéance et un statut (finie / à faire). 

Pour cela, le site disposera d'une interface de type formulaire qui comportera:
* Un input pour le titre
* Un input pour la date
* Une checkbox pour le statut

Chaque todo sera ensuite affiché dans un listing de type tableau, dans lequel on pourra visualiser le numéro de la tâche, son contenu, sa date d'échéance avec le nombre de jours restants, et son statut. Il sera possible, pour chaque tâche, de changer son statut ou de la supprimer si celle-ci est finie.
